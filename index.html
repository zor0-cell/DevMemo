⚙️ Требуется настройка

Для авто-синхронизации подключите бесплатный Firebase. Следуйте инструкции ниже.

Как настроить (бесплатно):

  1. 1 Зайдите на console.firebase.google.com → Создайте проект
  2. 2 В проекте → Build → Firestore Database → создайте базу в тестовом режиме
  3. 3 Настройки проекта (⚙) → Ваши приложения → добавьте веб-приложение (</>)
  4. 4 Скопируйте firebaseConfig и вставьте в начало скрипта (замените YOUR_...)
const FIREBASE_CONFIG = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT_ID.appspot.com",
  messagingSenderId: "YOUR_SENDER_ID",
  appId: "YOUR_APP_ID"
};

После вставки конфига сохраните файл и перезагрузите страницу — синхронизация включится автоматически.
